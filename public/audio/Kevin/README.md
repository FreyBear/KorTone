# Kevin vocal samples

Legg egne vokalsamples her for instrumentet Kevin (Vokal-sampler) i appen.

Obligatoriske filer:

- C4.mp3
- Ds4.mp3
- Fs4.mp3
- A4.mp3

I Kevin-modus brukes filene i denne mappen.
Denne mappen erstatter ikke Flygel-instrumentet eller filene i `public/audio/piano-lite`.

Hvis filene mangler her, vil Kevin-modus falle tilbake til Flygel-lyd.
